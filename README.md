# YOLO Result Repository

이 폴더는 UCGV(Unmanned Combat Ground Vehicle) 프로젝트에서 YOLO 학습 및 추론 결과를 저장하는 곳입니다.  

## 폴더 구조
- `12_06~12_18`: 각 날짜별 학습 결과 및 모델 파일
- `데이터 증강 코드`: 데이터 전처리 및 증강 스크립트
- `라스베리 파이 저장본`: 라즈베리파이에서 실행한 추론 코드
- `컴퓨터에서 학습된 모델 돌리기`: PC에서 학습된 모델을 이용한 추론 코드
- `Yolo hp설정.docx`: 학습 하이퍼파라미터 및 설정 문서
- `README.md`: 현재 문서

## 내용
- `.pt`: PyTorch 모델 파일
- `.tflite`: TensorFlow Lite 모델 파일
- `.csv`: 학습/추론 결과 데이터
- `.py`: 학습 및 추론 스크립트
- `.docx`: 관련 문서

## 사용법
1. 학습 모델을 불러와 추론 실행
2. 결과 `.csv` 확인 및 분석
3. 필요 시 데이터 증강 코드 사용

## 주의 사항
- 파일명 및 폴더명이 한글/특수문자를 포함할 수 있음
- 큰 파일은 Git LFS 사용
